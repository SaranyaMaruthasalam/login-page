🔐 Login Page : Project Overview
Secure Gateway is a clean and simple login page. It provides a neat, welcoming box where users can type in their username and password to log into a website. It is designed to look good and fit perfectly on both computer monitors and mobile phone screens.

🛠 Tech Stack & Architecture
HTML5: The skeleton of the page. It is used to put the actual boxes, text, and buttons on the screen.
CSS3: The paint and style. It is used to add colors, decide the spacing, change the fonts, and make things move smoothly.
How it was built: Everything was built entirely from scratch by hand, without using any pre-made design templates or automatic setup tools.

💡 Key Features Implemented
1. The Centered Box:
Instead of having the login inputs floating around randomly, everything is neatly placed inside a nice box right in the exact center of the screen.
How it works in code: I used a special alignment tool in CSS (called Flexbox) that forces the login box to stay perfectly centered, no matter how much you stretch or shrink the browser window.
2. Modern Dark Colors:
To make the page look modern and easy on the eyes, I used a dark theme instead of a plain, boring white background.
How it works in code: I picked a deep, dark purple/indigo color for the background. This makes the white text and input fields pop out clearly so they are very easy to see.
3. Customized Links
Standard internet links are usually bright blue with an underline, which can look a bit old-fashioned. I redesigned the "Sign up" link to look much better.
How it works in code: I removed the default underline, changed the text color to a nice bright indigo, and made the text bold so users can instantly see where to click if they need to create a new account.
4. Giving Elements "Breathing Room"
If buttons and text boxes are glued too close together, it looks messy and is hard to click.
How it works in code: I added strict spacing rules (margins) above the elements to push them apart vertically. This gives the page a clean, spaced-out look that is comfortable to read.
